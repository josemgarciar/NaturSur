NaturSur (Django)

Este proyecto usa Django. Para verlo en marcha, instala dependencias y arranca el servidor de desarrollo.

  1. Crear venv e instalar: pip install -r requirements.txt
  2. Aplicar migraciones: python manage.py migrate
  3. Arrancar: python manage.py runserver

Luego visita http://127.0.0.1:8000
